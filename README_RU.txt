# Закупки (PWA) — KompVLZ стиль

Тёмная PWA под iPhone: закупка → экспорт JSON → импорт в KompVLZsklad (позже).

## Быстрый старт (GitHub Pages)

### Вариант 1 (рекомендую): репозиторий + GitHub Pages (root)
1) Создай новый репозиторий на GitHub, например `zakupki`.
2) Закинь в него **все файлы из этого архива** (как есть, в корень репозитория).
3) GitHub → Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main` / `(root)`
4) Подожди пока появится ссылка вида:
   `https://<username>.github.io/zakupki/`

### Установка на iPhone
1) Открой ссылку в Safari.
2) Поделиться → **На экран Домой**.
3) Запускай как обычное приложение.

## Управление
- “+” — новая закупка
- Фильтр: Не импортированные / Все / Архив
- Внутри закупки:
  - ⤓ (короткое) — экспорт **этой** закупки
  - ⤓ (долгое) — backup **всех** закупок
  - ✅ — пометить как импортировано (уходит в архив, становится read-only)

## Ввод цен
В поле цены можно писать:
- `5000`
- `Цена - 5000`
- `5000р`
Если числа нет или пусто — цена = 0.

## Кол-во
Кол-во целое ≥ 1. Итог считается как `qty * price`.

## Хранение данных
Всё хранится в `localStorage` браузера (офлайн).

## Файлы экспорта
- `purchase_YYYY-MM-DD.json` — одна закупка
- `purchases_backup.json` — все закупки (backup)
